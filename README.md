#Pack Object

The `pack` object creates equal-width layouts.

	<div class="pack">
		<div class="pack__item"></div>
		<div class="pack__item"></div>
		<div class="pack__item"></div>
	</div>

## Installation

Install using Bower:

	$ bower install --save bare-pack

Import into project:

	@import "bower_components/bare-pack/objects.pack";
